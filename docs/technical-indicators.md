# 技術指標與訊號公式

所有指標皆由 TWSE 官方歷史行情在瀏覽器內依固定公式推導，來源標示為 `Derived`，不是 TWSE 官方結論，也不是 AI 生成。

## 指標

- SMA：MA5、MA10、MA20、MA60、MA120 的簡單移動平均。
- EMA：EMA12、EMA26，以首段 SMA 為種子後套用 `2 / (period + 1)`。
- RSI14：Wilder 平滑平均漲幅與跌幅。
- KD 9,3,3：9 日 RSV，K 與 D 初值 50、採 2/3 舊值與 1/3 新值。
- MACD 12,26,9：EMA12 - EMA26，訊號線為 DIF 的 EMA9。
- Bollinger 20,2：MA20 ± 2 倍 20 日母體標準差。
- ATR14：True Range 的 14 日平均。
- 量比：當日成交量 / 20 日平均成交量。
- 20／60 日報酬：最新收盤相對對應交易日前收盤。
- 20 日波動率：日報酬標準差乘以 `sqrt(252)` 年化。

不足期數時回傳 `null`，不可用 0 偽裝。技術訊號只在所需當日與前一日指標皆存在時觸發。
