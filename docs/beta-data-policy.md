# GULI 公開測試資料政策

GULI v0.8.0-beta.1 預設啟用 Public Beta Mode。核心頁面使用以下狀態：

- `Official`：欄位直接來自已驗證的 TWSE 官方盤後資料。
- `Mixed`：同頁同時包含官方、GULI 規則推導或 Mock 資料。
- `Mock`：資料是集中管理的功能測試資料。
- `Fallback`：正式來源不可用，改採明確標示的回退資料。
- `Stale`：資料可能不是最近合理交易日。
- `Missing`：欄位尚未取得，顯示 `null` 或「尚未取得」。

## 呈現原則

1. Mock 數值不得標示為官方行情。
2. Derived 分數不得宣稱為 TWSE 官方結論。
3. 缺值不使用 0 或虛構值填補。
4. 含 Mock 或推導資料的頁面顯示「本頁包含模擬或推導資料，僅供功能測試。」
5. 官方行情加規則推導的頁面顯示「行情欄位來自 TWSE 官方盤後資料；分析結果為 GULI 規則推導。」
6. 所有核心分析保留「不構成投資建議」聲明。

## 近似漲跌停

目前市場廣度使用收盤漲跌幅門檻推導，UI 僅顯示「漲幅 ≥ 9.5%」與「跌幅 ≤ -9.5%」，並標示「衍生統計」。它們不是證交所依升降單位與價格限制計算的官方漲停／跌停家數。
