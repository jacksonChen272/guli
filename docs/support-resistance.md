# 支撐壓力與趨勢結構

公式版本：`support-resistance-v1.0`。

價位候選來自 20／60 日高低、局部高低轉折、MA20、MA60、布林軌道與 ATR 安全區。相近價位依 ATR 與股價比例合併成區帶，強度由來源數、觸及次數與最近觸及日判定。支撐中心不得高於現價，壓力中心不得低於現價。

資料少於 20 日時不產生價位；少於 60 日時不產生 60 日區間價位。系統不會用固定百分比虛構缺少的支撐或壓力。

趨勢分類僅使用收盤價、MA5／20／60／120、均線斜率、20 日高低與 MACD，分為多頭排列、偏多結構、盤整、偏空結構、空頭排列與資料不足。本版本沒有 ADX 資料，因此不宣稱使用 ADX。
